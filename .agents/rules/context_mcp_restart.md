# Context: Инициализация Figma MCP (Restart)

**Текущая точка отсчета (Starting Point)**

Только что был произведен перезапуск (restart) Antigravity пользователем после внесения исправлений в конфигурацию `mcp_config.json` (замена `npx` на `npx.cmd` для обхода блокировки PowerShell Execution Policy в Windows).

## Для чего это сделано:
- Чтобы Antigravity получил доступ к **Figma MCP Server**.
- Чтобы агент мог прочитать актуальный проект EatPan и сопоставить его с макетом Figma.
- **Главная цель:** когда разные разработчики с разных аккаунтов вносят изменения в макет Figma, агент (Antigravity) сможет считывать эти изменения через MCP-инструменты и автоматически вносить соответствующие правки в код (обновлять верстку, копировать макет, изменять дизайн компонентов).

## Статус
С этого момента агент должен по умолчанию считать, что интеграция с Figma — это основной источник истины для User Interface. При любых задачах, связанных с UI/интерфейсом, агент обязан руководствоваться новым правилом `sync_with_figma.md`.
