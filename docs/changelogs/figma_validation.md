# Валидация правок (Figma vs Code)

Этот файл используется агентом (Antigravity) для фиксации всех расхождений между текущим локальным кодом (правки в DevTools, ручные изменения UI) и актуальным макетом Figma.

Разработчик должен изучить этот список и принять решение (Заверить): что именно мы переносим из Figma в код, а что — из кода в Figma.

## Текущие расхождения
*(Агент будет добавлять новые нестыковки ниже в формате: Название элемента / Путь к коду / Разница / Статус решения)*

---
**Пример записи:**
- **Элемент:** Птичка в меню (`.menu-checkmark`)
- **Расположение в коде:** `src/modules/recipe-book/components/ui/Menu.js`
- **Значение в Figma:** Цвет `#000000`, размер `16px`
- **Значение в коде:** Цвет `var(--brand-red)`, размер `18px`
- **Решение:** [Ожидает выбора: Вариант А (Figma->Код) или Вариант Б (Код->Figma)]
