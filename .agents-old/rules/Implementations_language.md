# Russian Language Requirement

The user prefers to receive all structured documentation in Russian.

When creating or updating artifacts such as `implementation_plan.md`, `task.md`, `walkthrough.md`, or any custom files meant for user review, **you MUST write them in Russian** (unless code syntax or English-specific technical terms strictly demand otherwise).

This applies to descriptions, headers, lists, summaries, and explanations.
